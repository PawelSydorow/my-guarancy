# Code Review

Use this skill when reviewing diffs, auditing a change, or checking whether a patch is safe to merge.

## Focus Areas

- Correctness
- Security and tenant isolation
- Contract and backward-compatibility risk
- Test coverage
- UI and spec consistency

## Output

- Findings first
- Most severe issues first
- Include file references and concrete fixes
