# DS Guardian

Use this skill when checking UI consistency, hardcoded colors, arbitrary spacing, or other design-system drift.

## Checks

- Semantic tokens over hardcoded status colors
- Standard typography scale
- Consistent backend page patterns
- Reusable components over one-off UI
